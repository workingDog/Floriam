#  Floriam

A basic app to identify a plant from a picture taken with the camera or from the Photos App. The app uses [Pl@ntNet API](https://my.plantnet.org/doc/api/openapi) to identify the plant in the picture. "The identification engine is based on most advanced deep learning technologies ..."

## Usage

First select the **gear icon** and add your **api-key** and save.

The **History** keeps only the last 10 identified pictures.

Multiple selections (up to 3) from the **Photos App** can used to identify **one** plant.  

When using the **Camera**, only the one picture is used to identify the plant.

-   A valid **API key** is required. Create an account at [Pl@ntNet](https://my.plantnet.org/) to obtain one. A free account can be used.
